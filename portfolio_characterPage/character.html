아기공룡 둘리

대한민국 서울특별시 도봉구 쌍문동의
소시민 고길동의 집에 갑자기 군식구로 들어온
빙하시대 냉동된 아기공룡 둘리,
타임 코스모스의 고장으로 지구에 불시착한 외계인 도우너,
서커스단에서 탈출한 타조 또치 등과
고길동의 일상을 코믹하게 그려낸 만화입니다.

둘리

 출생 : 1억여 년 전 지구
종족 : 공룡
거주지 : 도봉구 쌍문동
성격 : 사교적, 친화력 있음, 의연함
특기 : "호이호이!" 라고 외치면 초능력을 발휘

둘리는 헤어진 엄마와 떠나온 고향에 대한 그리움으로 마음 한편에 외로움을 품고 있는 아기공룡입니다.

도우너

 출생 : 깐따삐야 별
종족 : 외계인
거주지 : 도봉구 쌍문동
성격 : 다혈질, 단순함, 예의가 없음
특기 : "깐따삐야!”를 외치면 시공간 여행을 할 수 있음

도우너는 우주여행을 하던 중 타임코스모스가 고장나 고길동 집 앞마당에 떨어지게 되면서 둘리와 함께 지내게 된 외계인입니다.

희동이

 출생 : 대한민국
종족 : 인간
거주지 : 도봉구 쌍문동
성격 : 고집이 셈, 가끔 난폭해짐
특기 : 쪽쪽이 물기

희동이는 고길동의 조카로 모두에게 사랑받는 아이이며, 둘리에게 “형아”라고 부르는 귀염둥이입니다.

또치

 근무지 : 라스베이거스 서커스단
종족 : 아프리카 타조
거주지 : 도봉구 쌍문동
성격 : 자존심 강함, 새침함, 우아함
특기 : 춤추기, 아크로바틱, 접시돌리기 등 각종 묘기

또치는 가끔 새나 닭으로 오인 받아 자존심이 상할 때도 있지만, 둘리 일행 중 유일한 홍일점인 아프리카 출신 귀부인 타조입니다.

마이콜

 출생 : 대한민국
종족 : 인간
거주지 : 도봉구 쌍문동
성격 : 겁이 많음, 소심함
특기 : 가수지망생으로 노래를 좋아하지만 잘 부르지는 못함

마이콜은 마르고, 큰 키에 심한 곱슬머리와 까무잡잡한 피부, 두꺼운 입술의 독특한 외모를 가진 '마(馬)'씨 성의 토종 한국인입니다.

고길동

 출생 : 대한민국
종족 : 인간
거주지 : 도봉구 쌍문동
성격 : 책임감이 강함
특기 : 둘리 일당 내쫓기, 가족들 먹여 살리기, 둘리 일당 구박하기

고길동은 가족들을 사랑하고 직장에서 최선을 다하는 대한민국의 평범한 가장입니다.
